# 停车场管理系统 java71

## 项目概述

停车场管理系统是一款基于B/S架构和MVC模式的Web应用，采用前后端分离的技术实现。该系统致力于为用户提供便捷的停车服务，提高停车效率。

## 技术栈

- 前端：Vue
- 后端：Java、Spring Boot
- 开发环境：IDEA、JDK 1.8、Maven 3.6.1、MySQL 8.0、Node 14、Navicat 17

## 功能模块

### 普通用户

- 个人信息管理
- 预定停车位
- 缴费信息管理

### 管理员用户

- 用户信息管理
- 车辆信息管理
- 停泊车辆管理
- 车辆进出管理

## 系统角色

- 普通用户
- 管理员

## 运行环境

- 服务器：支持Java和MySQL运行的服务器
- 客户端：现代Web浏览器

## 部署步骤

1. 部署后端服务
2. 部署前端服务
3. 配置数据库
4. 启动应用
